# Matematica - Guia de estudio

Esta es una pagina web estatica pensada como material de estudio autonomo para temas de matematica preuniversitaria.

## Contenido

La guia esta dividida en cuatro unidades:

- Logica proposicional
- Conjuntos
- Trigonometria
- Vectores

Cada unidad tiene dos partes principales:

- **Teoria fundamental:** explicaciones, formulas, tablas y aclaraciones para entender el tema antes de resolver.
- **Ejercicios resueltos:** dos ejercicios por unidad, desarrollados paso a paso.

## Videos de apoyo

En el encabezado de cada ejercicio hay un boton **Ver video**. Ese boton abre en YouTube el video correspondiente al ejercicio, en una nueva pestana.

Hay 8 ejercicios en total:

- EJ 01 y EJ 02: Logica proposicional
- EJ 03 y EJ 04: Conjuntos
- EJ 05 y EJ 06: Trigonometria
- EJ 07 y EJ 08: Vectores

## Donaciones

La pagina incluye un cartel de donaciones que aparece despues de unos segundos. Tambien hay un boton **Donaciones** en la barra superior para volver a abrirlo cuando se quiera.

## Uso

Abrir `index.html` en el navegador. No requiere servidor ni contrasena: cualquier persona con el archivo o el enlace puede entrar directamente.

## Archivos

- `index.html`: pagina completa con estilos, contenido y comportamiento.
- `README.md`: descripcion basica del proyecto.
