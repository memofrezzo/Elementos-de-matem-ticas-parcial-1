Funciones Reales

Una función es una "máquina" que toma un número y devuelve otro número. Por cada valor que meto (x), sale un único valor (y = f(x)).

¿Qué es una función?

Una función f: A → B asigna a cada x ∈ A un único y ∈ B.
Se escribe y = f(x). La x es la variable independiente, la y es la variable dependiente.

f(x) = x² − 3x + 2

Dominio e Imagen

Dominio: todos los valores de x para los que f(x) está definida.
Imagen: todos los valores que realmente toma f(x).
Cuidado: las raíces piden ≥ 0 adentro, y las fracciones piden denominador ≠ 0.

Ceros, C⁺ y C⁻

C⁰ (ceros): f(x) = 0 → la curva corta el eje x.
C⁺ (positividad): donde f(x) > 0 → la curva está arriba del eje x.
C⁻ (negatividad): donde f(x) < 0 → la curva está abajo del eje x.

Función Par e Impar

Par: f(−x) = f(x) → simétrica respecto al eje y. Ej: f(x) = x²
Impar: f(−x) = −f(x) → simétrica respecto al origen. Ej: f(x) = x³

Función Cuadrática

f(x) = ax² + bx + c.
El vértice está en xᵥ = −b/(2a).
Si a > 0 la parábola abre hacia arriba; si a < 0, hacia abajo.
Las raíces se calculan con la fórmula resolvente.

Asíntotas

Horizontal (AH): y = L cuando x→±∞. La curva se "aplana" en una recta horizontal.
Vertical (AV): x = a cuando f(x) → ±∞. La curva "explota" en una recta vertical.
Oblicua (AO): y = mx + b cuando x→∞.

Ejercicio Completo — Análisis de Función

Vamos a analizar todo sobre una función cuadrática y una racional. ¡Expandí cada paso!

📈
Parte A — Función Cuadrática
Analizá completamente f(x) = x² − 4x − 5

Encontrá: dominio, imagen, vértice, raíces, C⁰, C⁺, C⁻, intervalos de crecimiento/decrecimiento e intersecciones con los ejes.

f(x) = x² − 4x − 5
Resolución paso a paso
1
Dominio + ver

f(x) = x² − 4x − 5 es una función polinómica. Los polinomios están definidos para todos los números reales, no tienen denominadores ni raíces que puedan "romperse".

Dom(f) = ℝ = (−∞ , +∞)
Para funciones polinómicas el dominio SIEMPRE es ℝ. Problema con dominio solo ocurre con fracciones (denominador = 0) o raíces cuadradas de número negativo.
2
Vértice de la parábola + ver

Identificamos los coeficientes: a = 1, b = −4, c = −5.

La abscisa del vértice es:

xᵥ = −b / (2a) = −(−4) / (2·1) = 4/2 = 2

Para la ordenada, reemplazamos x = 2 en f(x):

yᵥ = f(2) = 2² − 4·2 − 5 = 4 − 8 − 5 = −9

El vértice es el punto V(2 ; −9). Como a = 1 > 0, la parábola abre hacia arriba → el vértice es un mínimo.

El vértice es el punto más alto o más bajo de la parábola. Si a > 0 es el punto más bajo (mínimo). Si a < 0 es el punto más alto (máximo).
3
Imagen (rango) + ver

Como la parábola abre hacia arriba (a > 0), el valor mínimo es el vértice yᵥ = −9. La función crece hacia +∞.

Im(f) = [−9 , +∞)
Si a < 0 (parábola hacia abajo), la imagen sería (−∞ , yᵥ]. El corchete "[" significa que el valor se incluye.
4
Raíces (ceros) — C⁰ + ver

Igualamos f(x) = 0 y aplicamos la fórmula resolvente:

x² − 4x − 5 = 0 x = [−(−4) ± √((−4)² − 4·1·(−5))] / (2·1) x = [4 ± √(16 + 20)] / 2 = [4 ± √36] / 2 = [4 ± 6] / 2

Dos soluciones:

x₁ = (4 + 6)/2 = 10/2 = 5 x₂ = (4 − 6)/2 = −2/2 = −1

La parábola corta el eje x en los puntos (−1 , 0) y (5 , 0).

C⁰ = {−1 , 5}

5
Positividad C⁺ y Negatividad C⁻ + ver

La parábola abre hacia arriba. Las raíces son x = −1 y x = 5. Analizamos el signo de f(x) en cada intervalo:

IntervaloEjemplo xf(x)Signo
(−∞, −1)x = −24 + 8 − 5 = 7✅ C⁺
(−1, 5)x = 00 − 0 − 5 = −5❌ C⁻
(5, +∞)x = 636 − 24 − 5 = 7✅ C⁺
C⁺ = (−∞, −1) ∪ (5, +∞) C⁻ = (−1, 5)
La curva está por encima del eje x en los intervalos C⁺, y por debajo en C⁻. Entre las raíces, la parábola "baja" debajo del eje.
6
Crecimiento y decrecimiento + ver

Como la parábola tiene vértice en xᵥ = 2 y abre hacia arriba:

I_dec = (−∞, 2) ← va bajando hasta el vértice I_crec = (2, +∞) ← sube desde el vértice

El eje de simetría es la recta vertical x = 2. La parábola es simétrica respecto a esa recta.

7
Intersección con el eje y + ver

Para encontrar dónde la curva corta el eje y, reemplazamos x = 0:

f(0) = 0² − 4·0 − 5 = −5

La curva corta el eje y en el punto (0 , −5).

La intersección con el eje y es siempre f(0), el coeficiente independiente "c" del trinomio.
📊 Gráfico de f(x) = x² − 4x − 5
✅

Resumen completo

Dom = ℝ  |  Im = [−9, +∞)  |  Vértice = (2, −9)  |  C⁰ = {−1, 5}

C⁺ = (−∞,−1)∪(5,+∞)  |  C⁻ = (−1, 5)  |  Eje sim: x = 2

Forma factorizada: f(x) = (x+1)(x−5)
📉
Parte B — Función Homográfica (Racional)
Analizá g(x) = (2x + 1) / (x − 3)

Encontrá: dominio, imagen, asíntotas, ceros, C⁺, C⁻ y graficá.

g(x) = (2x + 1) / (x − 3)
Resolución paso a paso
1
Dominio + ver

El único problema es el denominador. El denominador no puede ser cero:

x − 3 ≠ 0 → x ≠ 3 Dom(g) = ℝ − {3} = (−∞, 3) ∪ (3, +∞)
2
Asíntota Vertical (AV) + ver

La asíntota vertical ocurre donde el denominador se anula:

x − 3 = 0 → x = 3

La recta vertical x = 3 es la Asíntota Vertical. Si nos acercamos a x = 3, g(x) tiende a ±∞.

La asíntota vertical es como una "pared invisible" que la curva nunca puede cruzar.
3
Asíntota Horizontal (AH) + ver

Para funciones homográficas (ambos polinomios de igual grado), la AH es el cociente de los coeficientes principales:

AH: y = 2/1 = 2

Verificación: lim(x→∞) (2x+1)/(x−3) = lim(x→∞) 2x/x = 2

La recta horizontal y = 2 es la Asíntota Horizontal.

La imagen de la función homográfica es Im(g) = ℝ − {a/c} = ℝ − {2}. La función nunca toma el valor 2.
4
Ceros y positividad/negatividad + ver

g(x) = 0 cuando el numerador = 0:

2x + 1 = 0 → x = −1/2

C⁰ = {−1/2}. Eso significa que g corta el eje x en (−1/2, 0).

Intersección con eje y: g(0) = (2·0+1)/(0−3) = 1/(−3) = −1/3. Punto: (0, −1/3).

Intervalog(x)
(−∞, −1/2)C⁻ (negativa)
(−1/2, 3)C⁺ (positiva)
(3, +∞)C⁺ (positiva)
📊 Gráfico de g(x) = (2x+1)/(x−3)
✅

Resumen

Dom = ℝ−{3}  |  Im = ℝ−{2}  |  AV: x = 3  |  AH: y = 2  |  C⁰ = {−1/2}

∞ Unidad 6

Límite y Continuidad

Un límite responde a: "¿a qué valor se acerca f(x) cuando x se acerca a cierto punto?" — aunque f(x) no esté definida exactamente ahí.

Idea intuitiva

lim(x→a) f(x) = L significa que a medida que x se acerca a "a" (sin llegar a ser "a"), f(x) se acerca a "L".

lim(x→a) f(x) = L

Límites laterales

Por derecha: x → a⁺ (valores mayores que a)
Por izquierda: x → a⁻ (valores menores que a)
El límite existe ↔ ambos laterales son iguales.

Indeterminaciones

0/0: factorizar o racionalizar.
∞/∞: dividir por la mayor potencia de x.
∞ − ∞: operar algebraicamente primero.

Continuidad en un punto

f es continua en x = a si se cumplen las 3 condiciones:
1. f(a) existe
2. lim(x→a) f(x) existe
3. lim(x→a) f(x) = f(a)

Discontinuidades

Evitable: el límite existe pero f(a) no está definida o no coincide.
Esencial: los límites laterales son distintos, o el límite es ±∞.

Límite notable

Un resultado fundamental que debés memorizar:

lim(x→0) sen(x)/x = 1

Ejercicio — Límites Variados
Calculá los siguientes límites y analizá continuidad

Cinco límites de distintas técnicas: directa, indeterminación 0/0, irracional, ∞/∞ y análisis de continuidad.

Límite 1 — Sustitución directa
1
lim(x→2) (x² + 3x − 1) + ver

f(x) = x² + 3x − 1 es un polinomio → definida y continua en todos lados. Simplemente sustituimos x = 2:

lim(x→2) (x² + 3x − 1) = (2)² + 3(2) − 1 = 4 + 6 − 1 = 9
Para polinomios y funciones racionales (con denominador ≠ 0), el límite es simplemente f(a). ¡Sustituí directo!
Límite 2 — Indeterminación 0/0 (factorización)
2
lim(x→3) (x² − 9) / (x − 3) + ver

Si sustituimos x = 3 directamente: (9−9)/(3−3) = 0/0. ¡Indeterminación! Necesitamos factorizar.

x² − 9 = (x − 3)(x + 3) ← diferencia de cuadrados lim(x→3) [(x−3)(x+3)] / (x−3)

Cancelamos el factor (x − 3) que aparece arriba y abajo (recordá que x ≠ 3, solo se acerca a 3):

= lim(x→3) (x + 3) = 3 + 3 = 6
En una indeterminación 0/0 siempre hay un factor que se cancela. Factorizá numerador y denominador.

Límite 3 — Función Irracional (conjugada)
3
lim(x→0) (√(x+1) − 1) / x + ver

Sustituimos x = 0: (√1 − 1)/0 = 0/0. Indeterminación. Con raíces, la técnica es multiplicar por la expresión conjugada.

La conjugada de (√(x+1) − 1) es (√(x+1) + 1):

= lim(x→0) [(√(x+1)−1) / x] · [(√(x+1)+1) / (√(x+1)+1)]

El numerador se convierte en diferencia de cuadrados: (√(x+1))² − 1² = x+1 − 1 = x

= lim(x→0) x / [x · (√(x+1)+1)]

Cancelamos la x:

= lim(x→0) 1 / (√(x+1)+1) = 1 / (√1+1) = 1/2
La conjugada de (√A − B) es (√A + B). Al multiplicar: (√A − B)(√A + B) = A − B². Esto elimina la raíz del numerador.
Límite 4 — En el infinito (∞/∞)
4
lim(x→∞) (3x² − 2x) / (x² + 5) + ver

Si sustituimos x = ∞: ∞/∞ — indeterminación. La técnica: dividir numerador y denominador por la mayor potencia de x que aparece (aquí es x²):

= lim(x→∞) [(3x²/x² − 2x/x²)] / [(x²/x² + 5/x²)] = lim(x→∞) [3 − 2/x] / [1 + 5/x²]

Cuando x → ∞, los términos con x en el denominador se van a 0: 2/x → 0, 5/x² → 0.

= (3 − 0) / (1 + 0) = 3/1 = 3
Regla rápida para lim(x→∞) P(x)/Q(x):
• Si gr(P) = gr(Q): el límite es el cociente de coeficientes principales.
• Si gr(P) < gr(Q): el límite es 0.
• Si gr(P) > gr(Q): el límite es ±∞.

En este caso ambos son grado 2: resultado = 3/1 = 3. Confirma la asíntota horizontal y = 3.

Límite 5 — Análisis de Continuidad
5
¿Es continua h(x) en x = 1? + ver

Sea:

h(x) = { 4x + 3 si x < 1 { x² + 6 si x ≥ 1

Verificamos las 3 condiciones:

Condición 1: ¿Existe h(1)?
h(1) = 1² + 6 = 7 ✅

Condición 2: ¿Existe el límite?
— Límite por izquierda (x → 1⁻): usamos la rama 4x + 3 → 4(1) + 3 = 7
— Límite por derecha (x → 1⁺): usamos la rama x² + 6 → 1 + 6 = 7

lim(x→1⁻) h(x) = 7 = lim(x→1⁺) h(x) → lim(x→1) h(x) = 7 ✅

Condición 3: ¿Coinciden?
lim(x→1) h(x) = 7 = h(1) ✅

✅ Las 3 condiciones se cumplen → h(x) es CONTINUA en x = 1.
✅

Resultados

1) 9  |  2) 6  |  3) 1/2  |  4) 3  |  5) Continua en x = 1

f′ Unidad 7

Derivadas

La derivada mide cómo cambia una función. Geométricamente es la pendiente de la recta tangente a la curva en un punto.

Definición

La derivada de f en x₀ es el límite del cociente incremental:

f′(x) = lim(Δx→0) [f(x+Δx)−f(x)] / Δx

Reglas básicas

Constante: D(k) = 0
Potencia: D(xⁿ) = n·xⁿ⁻¹
Constante × función: D(k·f) = k·f′
Suma/Resta: D(f±g) = f′±g′

Producto y Cociente

Producto: D(u·v) = u′v + uv′
Cociente: D(u/v) = (u′v − uv′) / v²

(u/v)′ = (u′v − uv′) / v²

Regla de la cadena

Para funciones compuestas f(g(x)):

[f(g(x))]′ = f′(g(x)) · g′(x)

Primero derivás la función "de afuera" evaluada en la "de adentro", y multiplicás por la derivada de "adentro".

Funciones especiales

D(eˣ) = eˣ
D(ln x) = 1/x
D(sen x) = cos x
D(cos x) = −sen x
D(tg x) = sec²x = 1/cos²x

Recta tangente y normal

En el punto P(x₀, f(x₀)):
Tangente: y = f′(x₀)·(x − x₀) + f(x₀)
Normal: y = [−1/f′(x₀)]·(x − x₀) + f(x₀)

D(√x) = 1/(2√x)

Ejercicio Completo — Derivación

Derivamos una función compleja usando múltiples reglas, hallamos recta tangente y normal, y analizamos los puntos críticos.

f′
Ejercicio — Derivación completa
Sea f(x) = x³ − 6x² + 9x + 1. Analizá completamente.

Calculá f′(x) y f″(x), encontrá los puntos críticos, determiná máximos/mínimos y hallá la ecuación de la recta tangente en x = 0.

f(x) = x³ − 6x² + 9x + 1
Derivación y análisis completo
1
Calculá f′(x) — Derivada primera + ver

Aplicamos la regla de la potencia a cada término (derivada de la suma = suma de derivadas):

f(x) = x³ − 6x² + 9x + 1 D(x³) = 3x² D(−6x²) = −6 · 2x = −12x D(9x) = 9 D(1) = 0 ← derivada de constante f′(x) = 3x² − 12x + 9
La derivada de cada término es independiente. Regla de la potencia: D(xⁿ) = n·xⁿ⁻¹. Bajamos el exponente y lo ponemos como coeficiente, luego reducimos el exponente en 1.
2
Calculá f″(x) — Derivada segunda + ver

Derivamos f′(x) para obtener f″(x):

f′(x) = 3x² − 12x + 9 D(3x²) = 6x D(−12x) = −12 D(9) = 0 f″(x) = 6x − 12
La derivada segunda nos dice cómo cambia la pendiente. Nos sirve para clasificar los puntos críticos: f″ > 0 → mínimo, f″ < 0 → máximo.
3
Puntos críticos (donde f′ = 0) + ver

Los puntos críticos son donde la derivada primera se anula (pendiente = 0, es decir la función "cambia de dirección"):

f′(x) = 0 → 3x² − 12x + 9 = 0

Dividimos todo por 3:

x² − 4x + 3 = 0

Factorizamos:

(x − 1)(x − 3) = 0 x₁ = 1 y x₂ = 3

Calculamos f en esos puntos:

f(1) = 1 − 6 + 9 + 1 = 5 → punto crítico: (1, 5) f(3) = 27 − 54 + 27 + 1 = 1 → punto crítico: (3, 1)
4
Clasificá los puntos críticos con f″ + ver

Evaluamos la derivada segunda en cada punto crítico:

f″(1) = 6(1) − 12 = 6 − 12 = −6 < 0 → MÁXIMO LOCAL en (1, 5) f″(3) = 6(3) − 12 = 18 − 12 = 6 > 0 → MÍNIMO LOCAL en (3, 1)
Regla del criterio de la segunda derivada:
• f″(x₀) < 0 → la función está "cóncava hacia abajo" → es un MÁXIMO LOCAL 🔝
• f″(x₀) > 0 → la función está "cóncava hacia arriba" → es un MÍNIMO LOCAL 🔻
Puntof″TipoValor
x = 1−6Máximo localf(1) = 5
x = 3+6Mínimo localf(3) = 1
5
Recta tangente en x = 0 + ver

La pendiente de la tangente en x = 0 es f′(0):

f′(x) = 3x² − 12x + 9 f′(0) = 3(0)² − 12(0) + 9 = 9 ← esta es la pendiente m

El punto de tangencia es (0, f(0)):

f(0) = 0 − 0 + 0 + 1 = 1 → P(0, 1)

Ecuación de la recta tangente y = m(x − x₀) + y₀:

y = 9(x − 0) + 1 → y = 9x + 1

Ecuación de la recta normal (pendiente perpendicular = −1/m):

y = (−1/9)(x − 0) + 1 → y = −x/9 + 1
La recta tangente "roza" la curva en ese punto. La recta normal es perpendicular a la tangente (pendientes inversas y de signo contrario: m₁ · m₂ = −1).

Ejercicios extra — Regla de la cadena y producto + ver

Ejemplo A: Derivá h(x) = sen(x²)

Función "de afuera": sen(u) → derivada: cos(u) Función "de adentro": u = x² → derivada: 2x h′(x) = cos(x²) · 2x = 2x·cos(x²)

Ejemplo B: Derivá k(x) = eˣ · ln(x) (producto)

u = eˣ → u′ = eˣ v = ln(x) → v′ = 1/x k′(x) = u′v + uv′ = eˣ·ln(x) + eˣ·(1/x) = eˣ[ln(x) + 1/x]

Ejemplo C: Derivá p(x) = (3x² + 1)⁵ (cadena con potencia)

u = 3x² + 1 → u′ = 6x p′(x) = 5(3x²+1)⁴ · 6x = 30x(3x²+1)⁴
En la regla de la cadena, siempre bajás el exponente exterior, dejás la función adentro sin cambiarla, y multiplicás por la derivada de adentro.

Cheat-Sheet: ¿qué regla uso?

Veo una suma o resta

  • Derivo término a término
  • D(f±g) = f′ ± g′

Veo una multiplicación

  • Regla del producto
  • D(u·v) = u′v + uv′

Veo una fracción

  • Regla del cociente
  • D(u/v) = (u′v − uv′)/v²

Veo función dentro de función

  • Regla de la cadena
  • D[f(g(x))] = f′(g(x))·g′(x)
